Dependências
------------------------------------------
gcc 4.8.4 (Ubuntu 4.8.4-2ubuntu1~14.04.1)


------------------------------------------
Instruções de uso
------------------------------------------

1. Compile o programa 
	$ gcc -o conversao_bases conversao_bases.c -lm
2. Execute o programa
	$ ./conversao_bases
3. Informe os dados solicitados pelo programa no próprio terminal, por exemplo:
	Digite um numero: 1101
	Digite a base de entrada: 2
	Digite a base de saida: 10
	-----------------------------
	Numero 1101(10) = 13
