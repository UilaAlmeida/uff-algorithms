# Dependências
* gcc 5.4.0


# Instruções de uso

1. Compile o programa
```
	$ gcc -o conversao_bases conversao_bases.c -lm
```

2. Execute o programa informando os dados solicitados, por exemplo:
``` 
	$ ./conversao_bases
    Digite um numero: 1101
    Digite a base de entrada: 2
    Digite a base de saida: 10
    -----------------------------
    Numero 1101(10) = 13
```
